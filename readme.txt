This plugin enables VexTab (http://www.vexflow.com/vextab/) functionality for WordPress (http://www.wordpress.org).

== Description ==

Simply put VexTab code inside [vextab][/vextab] pairs. Attributes can also be added.

VexTab-Shortcode is developed on
[Github](https://github.com/yxonic/vextab-shortcode).

== Installation ==

1. Unzip the downloaded .zip archive to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
